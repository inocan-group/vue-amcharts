# VennSeries

Creates Venn Diagram _Series_.

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/reference/vennseries/) on venn series for more details.

A `VennSeries` component is used in the default slot of an `VennDiagram` component to create a venn diagram.
