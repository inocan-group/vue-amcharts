# RadarColumnSeries

Defines _Series_ for a radar column graph.

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/reference/radarcolumnseries/) on radar column series for more details.

A `RadarColumnSeries` component is used in the default slot of a `RadarColumnChart` component to create a radar column graph.
