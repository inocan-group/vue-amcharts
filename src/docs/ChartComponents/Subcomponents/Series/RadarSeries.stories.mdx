# RadarSeries

Defines _Series_ for a radar graph.

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/reference/radarseries/) on radar series for more details.

A `RadarSeries` component is used in the default slot of a `RadarChart` component to create a radar graph.
