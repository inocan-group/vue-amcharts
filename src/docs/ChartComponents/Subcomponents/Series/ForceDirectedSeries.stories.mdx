# ForceDirectedSeries

Defines _Series_ for a `ForceDirectedTree` chart.

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/reference/forcedirectedseries/) on force directed series for more details.

A `ForceDirectedSeries` component is used in the default slot of a `ForceDirectedTree` component to create a force directed chart.
