# LineSeries

Defines _Series_ for a line chart (Displays a line graph. Can also be used to create an area graph).

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/reference/lineseries/) on column series for more details.

A `LineSeries` component is used in the default slot of an `XyChart` component to create a line chart.
