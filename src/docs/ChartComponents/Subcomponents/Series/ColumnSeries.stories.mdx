# ColumnSeries

Defines _Series_ for a column chart (displays columns or bars).

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/reference/columnseries/) on column series for more details.

A `ColumnSeries` component is used in the default slot of an `XyChart` component to create a column chart.
