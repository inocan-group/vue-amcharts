# PieSeries

"Defines Series for a slice series on a Pie chart."

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/reference/pieseries/) on pie series for more details.

A `PieSeries` component is used in the default slot of a `PieChart` component to create a pie chart.
