# XyCursor

An `XyCursor` component is used in the default slot of a [XyChart](?path=/docs/documentation-chart-components-xychart--page) component to create a cursor.

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/concepts/chart-cursor/) on cursors for more details.
