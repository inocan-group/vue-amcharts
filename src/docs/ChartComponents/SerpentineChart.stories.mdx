# SerpentineChart

"SerpentineChart is a wrapper for CurveChart which automatically generates horizontal or vertical waved curves, based on certain parameters. It sort of looks like a mountain road, hence the name."

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/chart-types/timeline/#Serpentine) on serpentine charts for more details.
