## How to compose a Word Cloud

A Word Cloud consists of a parent [WordCloud](?path=/docs/documentation-chart-components-wordcloud--page) component containing the following in its default slot:
.

- A [WordSeries](?path=/docs/documentation-chart-components-subcomponents-series-wordseries--page), used to define a list of keywords or text for the word cloud.
