# Pie Charts

"A Pie chart is a circular chart which is divided into slices to show relative sizes of data."

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/chart-types/pie-chart/) on pie charts for more details.

## How to compose a Pie Chart

A Pie chart consists of the following elements:

- A parent [PieChart](?path=/docs/documentation-chart-components-piechart--page) component containing the following in its default slot:
  - At least one [PieSeries](?path=/docs/documentation-chart-components-subcomponents-series-pieseries--page) component.
