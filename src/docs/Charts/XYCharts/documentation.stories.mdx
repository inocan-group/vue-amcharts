# Xy Charts

"We use XY chart to plot any kind of serial 2/3-dimensional data, including line, area, column, bar, candlestick, ohlc, stepline, even heatmaps. Basically, any data, that requires 2 dimensions can be depicted using XY chart"

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/chart-types/xy-chart/) on XY charts for more details.

## How to compose an Xy Chart

An XY chart consists of the following elements:

- A parent [XyChart](?path=/docs/documentation-chart-components-xychart--page) component containing the following in its default slot:
  - At least two axes - vertical and horizontal
  - At least one series component ([ColumnSeries](?path=/docs/documentation-chart-components-subcomponents-series-columnseries--page) for a column chart, [LineSeries](?path=/docs/documentation-chart-components-subcomponents-series-lineseries--page) for a line chart, [CandlestickSeries](?path=/docs/documentation-chart-components-subcomponents-series-candlestickseries--page) for a candlestick chart, etc)

### Axes

Please refer to the [ValueAxis](?path=/docs/documentation-chart-components-subcomponents-axes-valueaxis--page), [DateAxis](?path=/docs/documentation-chart-components-subcomponents-axes-dateaxis--page) and [CategoryAxis](?path=/docs/documentation-chart-components-subcomponents-axes-categoryaxis--page) pages for more information on how to configure axes.
