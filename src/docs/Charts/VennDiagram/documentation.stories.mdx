# Venn Diagrams

"Venn Diagrams are used to display scale of relationship between different items by showing a series of overlapping bubbles. The bigger overlapping area, the closer relation."

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/chart-types/venn/) on Venn diagrams for more details.

## How to compose a Venn Diagram

A Venn diagram consists of a [VennDiagram](?path=/docs/documentation-chart-components-venndiagram--page) component, containing the following in its default slot:

- A [VennSeries](?path=/docs/documentation-chart-components-subcomponents-series-vennseries--page) component, used to define data and data fields for the diagram.
