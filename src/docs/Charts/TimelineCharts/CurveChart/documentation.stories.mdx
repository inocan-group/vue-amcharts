## How to compose a Curve Chart

A Curve chart consists of the following elements:

- A parent [CurveChart](?path=/docs/documentation-chart-components-curvechart--page) component containing the following in its default slot:
  - At least two axes - vertical and horizontal
  - At least one series component ([CurveColumnSeries](?path=/docs/documentation-chart-components-subcomponents-series-curvecolumnseries--page) for a column chart, [CurveLineSeries](?path=/docs/documentation-chart-components-subcomponents-series-curvelineseries--page) for a line chart, or a [CurveStepLineSeries](?path=/docs/documentation-chart-components-subcomponents-series-curvesteplineseries--page) for a stepped line chart).

### Axes

Please refer to the [ValueAxis](?path=/docs/documentation-chart-components-subcomponents-axes-valueaxis--page), [DateAxis](?path=/docs/documentation-chart-components-subcomponents-axes-dateaxis--page) and [CategoryAxis](?path=/docs/documentation-chart-components-subcomponents-axes-categoryaxis--page) pages for more information on how to configure axes.
