# Force Directed Trees

"Force Directed Tree is a special kind of chart used to display multi-item data related in hierarchical, linear or mixed way, as a series of linked bubbles."

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/chart-types/force-directed/) on force directed trees for more details.

## How to compose a Force Directed Tree

A Force Directed Tree consists of a parent [ForceDirectedTree](?path=/docs/documentation-chart-components-forcedirectedtree--page) component (used to define data for the tree) containing the following in its default slot:
.

- A [ForceDirectedSeries](?path=/docs/documentation-chart-components-subcomponents-series-forcedirectedseries--page) component, used to define data fields for the tree.
