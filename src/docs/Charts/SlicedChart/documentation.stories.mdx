# Sliced Charts

"Sliced charts are used to illustrate some stepped process. With each steps the total initial value is gradually reduced or increased to make the impact of the step more prominent, than it would be possible with other relational types, like, say, Pie chart."

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/chart-types/sliced-chart/) on sliced charts for more details.

## How to compose a Sliced Chart

A Sliced chart consists of the following elements:

- A parent [SliceChart](?path=/docs/documentation-chart-components-slicechart--page) component (used to define the chart data) containing the following in its default slot:
  - At least one series component ([FunnelSeries](?path=/docs/documentation-chart-components-subcomponents-series-funnelseries--page), [PyramidSeries](?path=/docs/documentation-chart-components-subcomponents-series-pyramidseries--page), or [PictorialStackedSeries](?path=/docs/documentation-chart-components-subcomponents-series-pictorialstackedseries--page))
