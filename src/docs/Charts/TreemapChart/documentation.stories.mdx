# Treemap Charts

"TreeMap is a great way to represent distribution of values among hierarchical categories in a clear and compact format."

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/chart-types/treemap/) on treemap charts for more details.

## How to compose a Treemap Chart

A Treemap chart consists only of a [TreemapChart](?path=/docs/documentation-chart-components-treemapchart--page) component, used to define data and data fields for the chart.
