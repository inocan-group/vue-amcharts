# Chord Diagrams

"Chord diagram is a special chart type, used to indicate one-level quantitative relations between multiple items, organized in a circular diagram."

Please refer to [amChart's documentation](https://www.amcharts.com/docs/v4/chart-types/chord-diagram/) on chord diagrams for more details.

## How to compose a Chord Diagram

A Chord diagram consists of only a [ChordDiagram](?path=/docs/documentation-chart-components-chorddiagram--page) component, used to define data for the chart.
